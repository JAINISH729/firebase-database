Inventory Dashboard

The Inventory Dashboard is a web-based application designed to manage product inventory efficiently in real time. It allows users to add, view, update, and delete product records while maintaining synchronization with a Firebase Realtime Database.

Project Overview

This application helps businesses or individuals keep track of inventory items such as product name, category, quantity, and price. All operations are handled seamlessly with real-time updates, ensuring data consistency across the system.

Key Features

• Add Product
Users can add new inventory items with details such as name, category, quantity, and price.

• View Products
All products are displayed in a structured table format with clear and readable data.

• Update Product
Existing product details can be edited directly within the table using an inline editing feature.

• Delete Product
Products can be removed from the inventory when no longer needed.

• Real-Time Data Sync
All data is stored in Firebase Realtime Database and updates instantly without page refresh.

Technology Stack

• React.js – Frontend library for building user interfaces
• Redux Toolkit – State management for handling application data
• Redux Thunk – Middleware for asynchronous operations
• Firebase Realtime Database – Backend for storing and syncing data
